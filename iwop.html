소개

소프트웨어과 전문동아리 중 유일하게 웹동아리인 IWOP은 19년의 역사를 자랑하는 동아리 입니다. 우수한 수상실적을 보유하고있으며, 웹 클라이언트, 서버, 웹 게임 개발과 웹 디자인을 배울 수 있습니다!

커리큘럼

1. FrontEnd Basic

처음 IWOP에 들어오면 HTML5와 CSS3로 정적페이지를 만들어봅니다.

2. Javascript

자바스크립트로 동적인 홈페이지를 제작해봅시다.

3. Node.js

Node.js와 Express 프레임워크로 웹서버를 만들어봅시다.

4. Database, Server

MongoDB, Socket.io, Passport로 다양한 서버를 제작해봅니다.

5. FrontEnd Advanced

Vue로 SPA를 만들어 실제로 배포해봅니다.

6. Web Game

Phaser.js로 웹게임을 개발하고 기획해봅니다.
